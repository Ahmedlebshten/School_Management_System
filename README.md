Project Title:
- School Management System

Project Idea:
- A basic system to manage student data and academic records.
- It allows students to log in using their ID and class to view their personal information and their marks.
- Students can print their marks.

Project Significance:
- A practical application of Backend Development concepts using PHP and MySQL.
- It demonstrates secure database handling by separating sensitive information into a .env file.
- It follows organized project structure separating assets, source code, and environment configurations.

Requirements:
- PHP 7.4 or higher
- MySQL Server
- Composer
- Git (optional)
- Local server environment (XAMPP, MAMP)

Technologies Used:
- Native PHP
- MySQL
- Composer
- HTML
- CSS
- Dotenv (for environment variable management)
